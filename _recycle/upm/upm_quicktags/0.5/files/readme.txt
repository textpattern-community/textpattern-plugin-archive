+Install Instructions+

1. Install the plugin.

2. Place the folder, "upm_quicktags", and all its contents inside the "textpattern" folder.

3. Read the "help" documentation provided, to ensure you understand how to use the plugin.


+ Editable Files +

- style.css
- write.js
- forms.js
- pages.js
- custom.js (only modify this if you know what you're doing)

Please do not modify the other files, or the plugin may cease to work.
